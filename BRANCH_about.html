Инвестиционный Калькулятор

База данных создана в рамках проектной работы при поддержке АНО ДО "Роболатория", Гончаренко А.А. Обратившись к странице с данными, любой человек/организация сможет получить данные о ставке с суточной точностью.

API

Нужная информация получается через использование API (Application Programming Interface) программами разного рода.

https://tools-api.robolatoriya.com/cbrfkeyrate

Первый метод представлен выше. Он отдаёт актуальное значение ключевой ставки на данный момент.
Второй метод даёт доступ ко всему массиву данных о ключевой ставке ЦБ РФ по датам.
Примечание: по прошествии времени, данных в таблице становится больше. У некоторых программ могут возникнуть проблемы в обработке данных, если они не оптимизированы под работу с большими списками.

https://tools-api.robolatoriya.com/cbrfkeyrate/all

Третий метод требует ввода двух переменных - "amount" и "months". С его помощью можно расчитать вашу общую и ежемесячную прибыль при инвестиции в "Роболаторию". В калькулятор можно также перейти по ссылке.

https://tools-api.robolatoriya.com/cbrfkeyrate/invest
